# AutoHotkey

# Hotkeys 和 Hotstrings

;;; Hotkeys 脚本结构 ;;;

^j::        ; 快捷键写在 `::` 之前
Send, Hello ; 要执行的命令
return      ; 结束命令

;;; Hotstrings ;;;

::ftw::Free the whales  ; 输入 "ftw" 就会自动展开为 "Free the whales"

.ahk 脚本编译成 .exe 文件之后就不依赖 AutoHotkey 了

# 自动填写表单

#IfWinActive, ahk_class YourWinClass  ; 可以用 Window Spy（见托盘右键菜单）查看各个窗口的 Window Class 值

Send {Tab}
SendRaw ********  ; Password
Send {Tab 2}
Send {Space}
Send +{Tab}
return
