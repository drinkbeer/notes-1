# 梯度下降优化算法总结

# 背景

回顾一下，现实中我们有了数据集 D={(x(1),y(1)),⋯,(x(m),y(m))}D = \lbrace(x^{(1)},y^{(1)}),\cdots,(x^{(m)},y^{(m)})\rbraceD={(x(1),y(1)),⋯,(x(m),y(m))}，也定义了损失函数 ℓ ⁣:Y×Y→R\ell\colon\mathcal{Y}\times\mathcal{Y}\to\mathbb{R}ℓ:Y×Y→R，我们想要寻找一个假设函数 hhh，which 能最小化经验误差（hhh 的所有参数由 θ\thetaθ 表示）：

min⁡θE^(θ;D),where E^(θ;D)=1m∑i=1mℓ(h(x(i);θ),y(i)).\min_\theta\widehat{E}(\theta;D)\text{,}\qquad\text{where }\widehat{E}(\theta;D)=\frac{1}{m}\sum_{i=1}^m \ell\mathopen{}\left(h(x^{(i)};\theta), y^{(i)}\right)\mathclose{}. θmin​E(θ;D),where E(θ;D)=m1​i=1∑m​ℓ(h(x(i);θ),y(i)).

这个（凸）优化的过程就可以用梯度下降来做

θ=θ−η⋅g\theta = \theta - \eta\cdot g θ=θ−η⋅g

其中 η\etaη 为学习率，g=∇θE^(θ)g = \nabla_\theta\widehat{E}(\theta)g=∇θ​E(θ) 为梯度
